    Questao_02
//---------------

a.) x' . (x + y')'

Aplicando De Morgan:

( x + y′ )′ = x′ . ( y′ )′= x′ . y

Substituindo:

x′. (x′. y) = x′. y


Simplificado:


f = x′ . y

//---------------

b.) (x + y) + (x'. y)

Pela absorcao:

 ( x + y ) + ( x′ . y ) = ( x + y )
 ( pois  x + y já cobre todos os casos de x′. y)

Simplificado:
 f = x + y

//---------------

c.) (x'. y)'. (x + y)

De Morgan no primeiro termo:

 ( x′ . y )′ = ( x′ )′ + y′ = x + y′

Substituindo:

 ( x + y′ ) . ( x + y )

Usando distributiva:

 = x + ( y′ . y )
 = x + 0 
 = x


Simplificado:

 f = x

//---------------

 d.) ( x . y' )' + (x + y)

De Morgan no primeiro termo:
 (x . y′)′= x′ + ( y′ )′ = x′ + y

Substituindo:

 (x′ + y ) + ( x + y )

Absorcao:

(x′ + y ) + x = 1 + y = 1

 Simplificado:

f = 1

//---------------

e.) (y' + x') . (x + y)

Distribuindo (propriedade distributiva):

 ( y′ + x′ ) . ( x + y )
= ( y′ . ( x + y ) ) + ( x′. ( x + y ) )

Expandindo:

= ( y′ . x + y′ . y ) + ( x′ . x + x′ . y )

Simplificando:

= ( y′ . x + 0 ) + ( 0 + x′. y )

Simplificado:

 f = x . y′ + x′ . y ( XOR )
